Politique de Confidentialité de Easy Task

Dernière mise à jour : 05 Décembre 2025

Cette politique de confidentialité décrit comment l'application Easy Task (ci-après "l'Application"), développée par Yann Dipita, recueille, utilise et protège vos données personnelles.

1. Données collectées

Nous collectons un minimum de données pour assurer le bon fonctionnement du service :

A. Données fournies par l'utilisateur

  • Adresse Email : Utilisée pour la création de compte et l'authentification (via Firebase Auth).
  • Données de paiement : Si vous souscrivez à l'offre Premium, les transactions sont traitées de manière sécurisée par notre partenaire Stripe. Nous ne stockons jamais vos informations bancaires complètes sur nos serveurs.

B. Permissions de l'appareil

L'application peut demander l'accès à certaines fonctionnalités de votre appareil :

  • Caméra / Photos : Uniquement si vous choisissez d'ajouter une photo de profil. (NSCameraUsageDescription)
  • Notifications : Pour vous envoyer des rappels concernant vos échéances à venir.

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Vous fournir le service de gestion de tâches.
  • Gérer votre abonnement Premium.
  • Vous envoyer des notifications de rappel (si activées).
  • Améliorer la stabilité de l'application (via des rapports de crash anonymes).

3. Partage des données

Nous ne vendons jamais vos données personnelles à des tiers.

Vos données peuvent être partagées avec nos prestataires de services tiers de confiance, uniquement dans le but de faire fonctionner l'application :

  • Google Firebase : Hébergement de base de données, authentification et analyses.
  • Stripe : Traitement des paiements sécurisés.

4. Sécurité des données

Nous mettons en œuvre des mesures de sécurité conformes aux standards de l'industrie pour protéger vos données. Toutes les communications entre l'application et nos serveurs sont chiffrées (HTTPS/SSL).

5. Suppression des données

Vous avez le droit de demander la suppression complète de votre compte et de toutes les données associées.

Vous pouvez effectuer cette demande :

  • Directement depuis les paramètres de l'application (si l'option est disponible).
  • Ou en nous contactant par email à : dipitayq@gmail.com

6. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité. Toute modification sera signalée par une mise à jour de la date en haut de ce document.

7. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter à :

Email : dipitay@gmail.com
